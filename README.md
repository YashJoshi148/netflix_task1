# Data Analyst Internship - Task 1

## 📌 Project Overview
This project is part of my **Data Analyst Internship**. The objective of this task was to perform data cleaning and preprocessing to prepare the dataset for further analysis.

## 🎯 Objectives
- Clean and preprocess the dataset.
- Handle missing values effectively.
- Standardize date formats.
- Improve data quality for analysis.

## 🛠️ Data Cleaning Steps
The following preprocessing steps were performed:

- Filled missing values using:
  - **Forward Fill (`ffill`)**
  - **Backward Fill (`bfill`)**
- Removed rows containing remaining null values using `dropna()`.
- Standardized the **Date** column by changing its format in Microsoft Excel.
- Verified the dataset for consistency and completeness.
